## 🇬🇧 English

ScreenSwitcher is a lightweight Windows application that automatically adjusts your monitor refresh rate (Hz) depending on whether your device is plugged in or running on battery.

It is designed to improve battery life while keeping a smooth experience when connected to power.

### ✨ Features
- Automatic refresh rate switching (AC / Battery)
- Multi-monitor support
- Clean and minimal interface
- System tray integration
- Light and dark theme
- Very low CPU and RAM usage
- Native Windows API usage (no heavy dependencies)

### 🧠 How it works
ScreenSwitcher uses Windows system APIs to:
- Detect connected displays
- Read available display modes
- Apply refresh rate changes
- Monitor power state changes in real time

### 🎯 Use case
For example:
- 144 Hz when plugged in
- 60 Hz when on battery

## 🇪🇸 Español

ScreenSwitcher es una aplicación ligera para Windows que ajusta automáticamente los hercios (Hz) del monitor según si el equipo está conectado a la corriente o funcionando con batería.

Está pensada para mejorar la duración de la batería sin perder fluidez cuando el equipo está enchufado.

### ✨ Características
- Cambio automático de hercios (AC / batería)
- Soporte para múltiples monitores
- Interfaz limpia y minimalista
- Integración con la bandeja del sistema (tray)
- Tema claro y oscuro
- Consumo muy bajo de CPU y RAM
- Uso directo de APIs nativas de Windows

### 🧠 Cómo funciona
ScreenSwitcher utiliza APIs de Windows para:
- Detectar monitores conectados
- Leer modos disponibles
- Aplicar cambios de frecuencia
- Detectar cambios de energía en tiempo real

### 🎯 Ejemplo de uso
Por ejemplo:
- 144 Hz conectado a corriente
- 60 Hz en batería
